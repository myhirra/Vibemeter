Built for screenshots

The default layout is a dense 2x2 card that reads well on X, V2EX, GitHub issues, Discord, and team chat.

Redact before sharing

Use dashboard redact mode so project names, paths, transcript titles, and sensitive local detail stay out of public images.

No upload step

The card is rendered from local dashboard data. Vibemeter does not upload prompts, transcripts, project names, or usage history.

How to generate one

  • Install Vibemeter and let it read existing local Claude Code, Codex, or Cursor sessions.
  • Open the dashboard at http://localhost:9527.
  • Choose period and tool, then export the recap card from the usage panel.

Useful when

  • You want a public proof-of-work card without leaking project data.
  • You are sharing an AI coding workflow, cost story, or cache-hit-rate insight.
  • You need a small visual hook for a launch post or internal update.
